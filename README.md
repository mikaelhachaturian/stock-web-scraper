# Stock Market Python Web Scraper CLI

```shell
usage: stock_web_scraper.py [-h] [-t TIME] [-l] [-s STOCK_SYMBOL]

Stock market web scraping tool.

optional arguments:
  -h, --help            show this help message and exit
  -t TIME, --time TIME  time (in seconds) to check stocks in loop.
  -l, --list            list all prices for stocks. (from stock_symbols.txt)
  -s STOCK_SYMBOL, --symbol STOCK_SYMBOL
                        specific stock to get info about.
```
